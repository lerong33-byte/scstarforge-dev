The missile rack tree structure is organized hierarchically. At the top level, under the "MISSILES" category, there is one row for each missile rack hardpoint on the ship. These rows display the rack's model (e.g., MSD-514, MSD-423), its size (e.g., S5, S4), its capacity and currently equipped missile type (e.g., 4/4 Rattler II), and its value in aUEC.

Therefore, there is exactly one MSD row per rack hardpoint.

When a rack row is expanded, it reveals individual rows for each missile loaded into that rack. Each of these sub-rows shows the details of a single missile, including its icon, name (e.g., Rattler II, Thunderbolt III), size (e.g., S2, S3), type (e.g., Strike), its status ("Equipped"), and its individual value in aUEC.

The expand/collapse functionality is controlled by clicking on the main rack row. A chevron icon on the left indicates the state: pointing down when expanded and pointing right when collapsed. Clicking the row toggles this state, either revealing or hiding the individual missile rows beneath it.
